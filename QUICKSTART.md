# Быстрый старт

## Запуск Backend

1. Перейдите в папку backend:
```bash
cd backend
```

2. Создайте виртуальное окружение (рекомендуется):
```bash
python -m venv venv
venv\Scripts\activate  # Windows
# или
source venv/bin/activate  # Linux/Mac
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Убедитесь, что файл `.env` содержит правильные настройки подключения к MongoDB

5. Запустите сервер:
```bash
python run.py
```

Backend будет доступен на `http://localhost:8000`
API документация: `http://localhost:8000/docs`

## Запуск Frontend

1. Перейдите в папку frontend:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение:
```bash
npm start
```

Frontend будет доступен на `http://localhost:3000`

## Первые шаги

1. Откройте `http://localhost:3000/docs` для просмотра API документации
2. Используйте API для создания первого пользователя (администратора)
3. Войдите в систему через фронтенд
4. Создайте классы, предметы и других пользователей через API

## Пример создания администратора через API

```bash
POST http://localhost:8000/api/auth/register
Content-Type: application/json

{
  "username": "admin",
  "email": "admin@school.ru",
  "full_name": "Администратор",
  "password": "admin123",
  "role": "admin"
}
```

## Структура данных

### Роли пользователей:
- `student` - Ученик
- `teacher` - Учитель  
- `parent` - Родитель
- `admin` - Администратор

### Основные сущности:
- **Пользователи** - ученики, учителя, родители, администраторы
- **Классы** - школьные классы (например, "5А", "10Б")
- **Предметы** - учебные предметы
- **Оценки** - оценки учеников по предметам
- **Домашние задания** - задания для классов
- **Расписание** - расписание уроков

## Примечания

- Убедитесь, что MongoDB запущена и доступна по указанному адресу
- Для работы фронтенда должен быть запущен backend
- CORS настроен для работы с localhost:3000 и localhost:5173
